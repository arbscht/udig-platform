Graticule Decoration¶

Graticule decorations displays a grid aligned with longitude and latitude lines defined by a geographic coordinate system.

Styles¶

Graticule font, lines and projection (geographic coordinate system) are configurable. You can change them by opening the Style Editor dialog and select the style you want to change.

Graticule font can be styled.
